# Sistema de Gestão de Estoque em C#

Este é um projeto de console desenvolvido como parte do curso "Introduction to Programming With C#". O objetivo é gerenciar produtos, preços e quantidades sem o uso de Orientação a Objetos, focando em lógica procedural.

## Funcionalidades
- Cadastro de produtos (Nome, Quantidade).
- Listagem completa do estoque.
- Uso de arrays/listas paralelas para gerenciamento de dados.

## Conceitos C# Aplicados
- **Estruturas de Controle:** `switch` para o menu e `if-else` para validações.
- **Laços de Repetição:** `while` para manter o programa ativo e `foreach` para percorrer as listas.
- **Coleções:** Uso de `List<T>`.
- **Métodos:** Modularização das funções do sistema.
